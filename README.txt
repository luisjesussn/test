## Requisitos

- Docker
- Docker Compose

## Configuración

Antes de comenzar, asegúrate de tener Docker y Docker Compose instalados en tu sistema.
El proyecto estará disponible en la URL configurada en VIRTUAL_HOST. Puedes acceder a él en tu navegador.

1. Clona este repositorio en tu máquina local:

    Crea un archivo .env en la raíz del proyecto y configura las variables de entorno necesarias. Puedes utilizar el archivo .env.example como referencia.

    Ejecuta el siguiente comando para construir y ejecutar los contenedores:

    docker-compose up -d
